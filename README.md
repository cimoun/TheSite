# ToDo Лист

Простое и элегантное веб-приложение для управления задачами.

## Возможности

- ✅ Добавление новых задач
- ✅ Отметка задач как выполненных
- ✅ Удаление задач
- ✅ Фильтрация задач (все, активные, завершённые)
- ✅ Сохранение данных в localStorage
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Красивый современный интерфейс

## Использование

Просто откройте `index.html` в браузере. Никаких дополнительных зависимостей или установки не требуется!

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- localStorage API
