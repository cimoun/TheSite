Animation Details

All components use Framer Motion for smooth, professional animations:

  • Checkbox: Spring animation (stiffness: 500, damping: 30)
  • Task Items: Stagger animation with 50ms delay between items
  • Modal: Scale and fade animations on enter/exit
  • Buttons: Scale transforms on hover and tap
  • Empty State: Pulsing icon animation
