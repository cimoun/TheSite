# Assets Directory

This directory contains static assets for the website.

## swirl.png

The hero section background uses `swirl.png` when available. If this file is not present, the hero section will gracefully fall back to a gradient background using the design system colors (--leaf-1 and --leaf-2).

To add the swirl.png file:
1. Place your swirl.png image in this directory
2. The background will automatically use it via the CSS url('assets/swirl.png')
